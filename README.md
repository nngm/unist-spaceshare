# UNIST SpaceShare

## For Contributors...

Before getting start, please add `.gitmessage.txt` as your git template. Use the command below.

```
git config --global commit.template .gitmessage.txt
```

## To Execute...

If you pulled this repository, Enter below:

```
bash run.sh
```

Or, Enter this.

```
cd unist-spaceshare
npm run dev
```
